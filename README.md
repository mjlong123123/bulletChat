# OBS弹幕编辑网页程序产品介绍文档

# 一、产品概述

本产品是一款运行于浏览器端的OBS弹幕编辑工具，无需安装额外插件，通过WebSocket协议与OBS Websocket服务器建立实时连接，实现弹幕的快速编辑、预览与OBS场景同步播放功能。产品聚焦于简化直播/录播过程中的弹幕操作，支持多弹幕有序播放、场景精准匹配及弹幕样式自定义，为用户提供高效、便捷的弹幕控制体验，适用于主播、内容创作者等各类需要在OBS中添加弹幕的场景。

核心定位：浏览器端轻量化OBS弹幕编辑工具，实现“编辑-预览-同步”一站式操作，降低弹幕制作与投放门槛，提升内容呈现质感。

# 二、核心功能

## 2.1 弹幕基础编辑与投放

产品支持文本式弹幕的快速编辑与添加，用户可在网页端输入弹幕内容，点击添加按钮即可将弹幕同步至OBS对应场景中。弹幕播放遵循“右侧进入、左侧滑出”的逻辑，当弹幕完全移动出屏幕左侧后，会自动删除，避免屏幕冗余，确保弹幕播放的整洁性。

同时支持多弹幕批量添加，多条弹幕将按照添加顺序依次从屏幕右侧进入，有序滑出，实现多弹幕不重叠、流畅播放的效果，满足用户批量投放弹幕的需求。

## 2.2 实时预览功能

网页端内置弹幕预览画面，用户添加弹幕、调整弹幕样式及播放速度后，可实时在预览画面中查看弹幕的显示效果、移动轨迹，无需切换至OBS界面即可完成效果确认。预览画面与OBS实际显示效果完全一致，帮助用户快速调整参数，提升操作效率，减少反复切换的麻烦。

## 2.3 OBS场景同步匹配

产品可自动识别OBS中已添加的所有场景，并在网页端展示场景列表，用户可自由选择目标场景。选择完成后，添加的所有弹幕将仅在该场景中播放，切换OBS场景后，弹幕将自动暂停对应场景的播放，切换回目标场景后恢复播放，实现弹幕与场景的精准绑定，适配多场景直播/录播的弹幕投放需求。

该功能依托WebSocket协议的实时通信能力，确保场景选择与弹幕投放的同步性，无需手动在OBS中进行复杂设置，简化操作流程。

## 2.4 弹幕样式自定义

用户可根据自身内容风格，自由调整弹幕的核心样式参数，具体支持：

- 字体大小：可灵活调节弹幕文本的尺寸，适配不同屏幕分辨率及内容呈现需求；

- 字体颜色：支持自定义弹幕颜色，可选择纯色、渐变等多种样式（根据实际产品支持范围调整），提升弹幕视觉辨识度；

- 移动速度：可调节弹幕从右侧进入、左侧滑出的移动速度，适配不同的内容节奏，避免弹幕过快无法看清或过慢影响画面呈现。

## 2.5 WebSocket实时连接

产品通过WebSocket协议与OBS Websocket服务器建立稳定连接，实现弹幕数据的实时传输。连接过程简单便捷，无需复杂的配置步骤，只需确保OBS Websocket服务器开启，即可完成网页与OBS的快速联动，传输延迟低，确保弹幕投放与预览的实时同步，避免出现卡顿、延迟等问题。

该连接方式遵循OBS Websocket的通信规范，支持主流版本的OBS软件，兼容性强，无需担心版本不匹配的问题。

# 三、产品优势

## 3.1 轻量化便捷操作

运行于浏览器端，无需安装任何客户端或插件，打开网页即可使用，适配各类主流浏览器（Chrome、Edge、Firefox等），随时随地可进行弹幕编辑与控制，降低用户使用门槛。

## 3.2 实时联动，所见即所得

WebSocket实时连接技术，实现网页预览与OBS实际显示效果完全同步，用户在网页端调整的所有参数（样式、速度、场景），均可实时反馈至OBS，无需反复切换界面确认，提升操作效率。

## 3.3 场景精准匹配，适配多场景需求

自动识别OBS场景并支持自由选择，弹幕仅在目标场景播放，解决多场景切换时弹幕混乱的问题，适配直播、录播等多种场景下的弹幕投放需求，提升内容专业性。

## 3.4 样式灵活自定义，满足个性化需求

支持字体大小、颜色、移动速度的全方位自定义，用户可根据自身内容风格、屏幕尺寸，打造专属弹幕样式，提升内容的视觉表现力，增强观众观看体验。

## 3.5 多弹幕有序播放，整洁不冗余

多条弹幕按添加顺序依次播放，滑出屏幕后自动删除，避免弹幕堆积、重叠，确保画面整洁，同时满足用户批量投放弹幕的需求，适配不同内容节奏。

# 四、适用场景

- 直播场景：主播可实时添加互动弹幕、通知类弹幕（如活动提醒、福利通知），提升直播互动性；

- 录播场景：内容创作者可在后期编辑时，为视频添加解说类、标注类弹幕，增强视频可读性；

- 线上活动场景：线上会议、讲座等场景中，可添加提示类、强调类弹幕，辅助内容传递；

- 新手入门场景：无需掌握复杂的OBS操作，通过网页端简单操作即可完成弹幕投放，适合OBS新手使用。

# 五、使用流程（简易版）

1. 开启OBS软件，并启用OBS Websocket服务器（确保服务器处于可连接状态）；

![obs](https://github.com/mjlong123123/bulletChat/blob/main/images/obs.webp?raw=true)

2. 打开本弹幕编辑网页，完成与OBS Websocket服务器的连接  ；

![obs](https://github.com/mjlong123123/bulletChat/blob/main/images/connect.webp?raw=true)

3. 在网页端选择需要投放弹幕的OBS场景；

![obs](https://github.com/mjlong123123/bulletChat/blob/main/images/scene.webp?raw=true)

4. 编辑弹幕内容，调整字体大小、颜色、移动速度等参数；

5. 在网页预览画面中确认弹幕效果，点击“添加”按钮，弹幕将同步至OBS对应场景播放；

![obs](https://github.com/mjlong123123/bulletChat/blob/main/images/bullet.webp?raw=true)

6. 可继续添加多条弹幕，实现有序播放，弹幕滑出屏幕后自动删除。

![obs](https://github.com/mjlong123123/bulletChat/blob/main/images/obs_bullet.webp?raw=true)

# 六、注意事项

- 使用前请确保OBS软件已安装并开启，且OBS Websocket服务器处于正常运行状态，否则无法建立连接；

- 建议使用主流浏览器最新版本，避免因浏览器兼容性问题导致功能异常；

- 添加多条弹幕后，若出现播放卡顿，可适当降低弹幕移动速度或减少同时播放的弹幕数量；

- 弹幕滑出屏幕后将自动删除，无法恢复，若需重复播放，需重新添加该弹幕。

# 七、产品总结

本OBS弹幕编辑网页程序，以“轻量化、高效率、易操作”为核心，通过WebSocket实时联动OBS，实现弹幕编辑、预览、场景匹配、样式自定义等一站式功能。无需复杂配置，无需安装额外软件，帮助用户快速完成弹幕投放，提升内容呈现效果，适用于各类需要在OBS中添加弹幕的场景。无论是直播主播、内容创作者，还是OBS新手，都能通过本产品轻松实现弹幕控制，降低操作成本，提升创作效率。
